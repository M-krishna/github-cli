# github-cli
#### Fetching details about users from Github, Implemented using goroutines.

```
go run main.go -u (usernames)
```


```
Usage: [options]    
Options:  
  -u string  
        Search User (Eg. john,foo,bar) 
```

## Details about the user.
- Login
- Avatar
- URL
- Name
- Company
- Location
- Bio
- PublicRepositories
- Followers
- Following
